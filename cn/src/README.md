# 智能体系统系列

> 本书翻译自 [The Agentic Systems Series](https://gerred.github.io/building-an-agentic-system/)（英文原版），作者 [Gerred Dillon](https://github.com/gerred)。

欢迎阅读构建真正能在生产环境中运行的 AI 编程助手的完整指南。这个全面的三册系列将带你从基础概念到实现企业级协作系统。

## 关于本系列

你是否好奇过现代 AI 编程助手到底是如何工作的？在提示词和演示之外，有一套丰富的模式、架构和工程决策使这些系统真正有效。

本系列揭示了这些模式。这是一份缺失的文档——基于真实生产系统的实用工程指南，包括对 [Amp](https://ampcode.com)（协作平台）、[Claude Code](https://claude.ai/chat)（Anthropic 的本地 CLI 工具）以及 anon-kode 等开源实现的深入分析。

## 三册内容

### 第一册：构建智能体系统
*打好基础*

深入实践如何构建你的第一个 AI 编程智能体。本书分析真实实现以提取核心模式：

- **核心架构** - 基于 Ink/Yoga 的响应式 UI、流式响应和状态管理
- **工具系统** - 用于文件操作、代码执行和外部集成的可扩展架构
- **权限系统** - 在安全和效率之间取得平衡的安全模型
- **并行执行** - 无竞态条件的并发操作
- **命令系统** - 斜杠命令、上下文帮助和用户配置
- **实现模式** - 来自 Amp 和 Claude Code 架构的经验教训

适合准备构建超越简单聊天机器人的生产级编程助手的工程师。

[**开始阅读第一册 →**](introduction.md)

### 第二册：升级智能体系统
*从本地到协作*

将单用户智能体转化为企业级协作平台。基于对生产系统的深入分析：

- **可扩展架构** - 会话管理、状态同步和大规模性能优化
- **认证与身份** - OAuth 流程、凭证管理和多环境支持
- **协作模式** - 实时共享、团队工作流和并发编辑策略
- **企业特性** - SSO 集成、用量分析和合规框架
- **高级编排** - 多智能体协调、自适应资源管理和成本优化
- **生产策略** - 部署模式、迁移框架和真实案例研究

对于将 AI 助手从原型扩展到生产协作环境的团队来说，这是必读内容。

[**继续阅读第二册 →**](second-edition/README.md)

### 第三册：智能体系统的上下文化
*高级工具与上下文*

深入探讨智能体系统的高级工具系统和上下文管理。本书涵盖：

- **工具系统架构** - 添加新能力的可扩展框架
- **命令系统设计** - 斜杠命令、上下文帮助和配置
- **上下文管理** - 理解和维护会话上下文
- **实现深度剖析** - 真实世界的工具系统实现和模式

适合构建复杂智能体能力和上下文感知系统的工程师。

[**探索第三册 →**](arming-introduction.md)

## 前置条件

- 熟悉系统设计概念
- 基本了解 AI/LLM 集成
- 有 TypeScript/Node.js 或类似后端技术的经验
- 了解终端/CLI 应用（有帮助但非必须）

## 内容包括

本系列提供：

- **架构模式** - 来自生产 AI 编程助手的经验证设计
- **实现策略** - 应对常见挑战的实用方法
- **决策框架** - 何时使用不同模式及其权衡
- **代码示例** - 说明性的实现（为广泛适用性而通用化）
- **案例研究** - 真实世界的部署场景和经验教训

内容基于对生产系统的深入分析，模式经过提取和通用化以供你自己的实现使用。
